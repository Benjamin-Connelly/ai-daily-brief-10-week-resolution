# Week 1 – Resolution Tracker (PWA)

## Objective
Build and deploy a simple resolution tracker that will be used to track progress across all 10 weeks.

## Definition of Done
- Displays Weeks 1–10
- Tracks status per week
- Allows notes per week
- Persists data locally
- Deployable via public URL
- Installable as a PWA

## Explicit Non-Goals
- No authentication
- No backend
- No AI calls
- No automation
- No database

This project prioritizes shipping speed and future extensibility.
